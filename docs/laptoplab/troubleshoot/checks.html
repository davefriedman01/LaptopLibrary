Quick Checks¶

Shell

  • Which shell is currently set as the default?

    • echo $SHELL or echo $0

  • Is Bash installed? Which version?

    • bash --version

  • Is Zsh installed? Which version?

    • zsh --version

Environment Variables

  • echo $PATH (macOS & Linux)

  • echo %PATH% (Windows)

Python

  • Is Python installed? Which version?

    • python -V (macOS & Linux)

    • py -V (Windows)

  • Which Python installation is currently set as the default?

    • which python && which python3 (macOS & Linux)

    • where python && where python3 (Windows)

Anaconda

  • Is conda installed? Which version?

    • conda -V

Git

  • Is Git installed? Which version?

    • git --version

macOS

  • Is Homebrew installed? Which version?

    • brew -V
