  • tar “tape archive”

    • tar cvf archive.tar myDirectory Create a full backup of myDirectory.

    • tar cvvf archive.tar myDirectory Create a full backup of myDirectory with more verbose output.

    • tar cvvfz archive.tar.gz myDirectory Backup and compress myDirectory.

    • tar xvvfz archive.tar.gz Extract everything from archive.tar.gz.

    • tar xvvfz archive.tar.gz -C newDirectory --wildcards "*.csv" Extract all CSV files from archive.tar.gz and place them into newDirectory.

    • man tar How do I use command tar?

    • To use GNU tar on macOS, install it with brew install gnu-tar && gtar --version.

      • tar --version Check the version of BSD tar.

      • gtar --version Check the version of GNU tar.
