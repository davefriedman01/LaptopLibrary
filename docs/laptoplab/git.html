Commands¶

git branch

  • git branch

    • git branch -M main Rename the current branch to main.

    • git branch -av View all local and remote branches.

git checkout

  • git checkout

git config

  • git config How do I use command git config?

    • git config --global -l List the variable-value pairs in config file ~/.gitconfig.

    • git config --global user.name myUsername Set the commit name to myUsername.

    • git config --global user.email "myEmail@example.com" Set the commit email address to myEmail@example.com.

    • git config --global init.defaultBranch main Set the default branch name to main for new repos.
