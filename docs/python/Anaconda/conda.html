Commands¶

conda

  • conda -h How do I use command conda?

  • conda -V What is the conda version?

  • conda update conda && conda update anaconda Update Anaconda.

conda activate

  • conda activate -h How do I use command conda activate?

  • conda activate ENVNAME Activate conda environment ENVNAME.

conda clean

  • conda clean -h How do I use command conda clean?

  • conda clean --all Remove unused cache files including unused packages.

conda config

  • conda config -h How do I use command conda config?

  • conda config --add channels CHANNELNAME Add a channel to your conda configuration.

  • conda config --show Examine conda configuration.

  • conda config --show-sources Examine conda configuration services.

conda create

  • conda create -h How do I use command conda create?

  • conda create --name ENVNAME python=3.6 "PKG1>7.6" PKG2 Create a new environment named ENVNAME with specific version of Python and packages installed.

  • conda create --clone ENVNAME --name NEWENV Make an exact copy of an environment.

conda deactivate

  • conda deactivate -h How do I use command conda deactivate?

  • conda deactivate Deactivate the current environment.

conda env

  • conda env -h How do I use command conda env?

  • conda env list or conda info -e Which conda environments exist?

  • conda env export --name ENVNAME > envname.yml Export an environment to a YAML file.

  • conda env create --file envname.yml Create an environment from a YAML file.

  • conda env create Create an environment from the file named environment.yml in the current directory.

  • conda create --name NEWENV --file pkgs.txt Create an environment based on exact package versions.

conda info

  • conda info -h How do I use command conda info?

  • conda info Verify conda is installed and check the version number.

  • conda info --base Where is the base conda environment located in the filesystem?

  • conda info --envs

  • conda info --system

conda install

  • conda install -h How do I use command conda install?

  • conda install --name ENVNAME --revision REV_NUMBER Restore an environment to a previous revision.

  • conda install conda-forge::PKGNAME Install a package from a specific channel.

  • conda install PKGNAME==3.1.4 Install a package by exact version number.

  • conda install "PKGNAME[version='3.1.2|3.1.4']" Install one of the listed versions (OR).

  • conda install "PKGNAME>2.5,<3.2" Install following several constraints (AND).

  • conda install --yes PKG1 PKG2 Run most commands without requiring a user prompt.

conda list

  • conda list -h How do I use command conda list?

  • conda list List all packages and versions in the active environment.

  • conda list -n ENVNAME List all packages and versions in environment ENVNAME.

  • conda list --revisions List all revisions made within the active environment.

  • conda list -n ENVNAME --revisions List all revisions made within environment ENVNAME.

  • conda list --explicit > pkgs.txt Create an environment based on exact package versions.

conda remove

  • conda remove -h How do I use command conda remove?

  • conda remove --name ENVNAME --all Delete an entire environment.

conda search

  • conda search -h How do I use command conda search?

  • conda search PKGNAME --info Detailed information about package versions.

conda uninstall

  • conda uninstall -h How do I use command conda uninstall?

  • conda uninstall PKGNAME --name ENVNAME Remove a package from an environment.

conda update

  • conda update -h How do I use command conda update?

  • conda update conda

  • conda update -n base conda Update conda to the current version.

  • conda update anaconda Update all packages to the latest version of Anaconda.

  • conda update --all --name ENVNAME Update all packages within an environment.

anaconda

  • anaconda search NAME Find a package on all channels using the Anaconda client.
