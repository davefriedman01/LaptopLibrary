Internal Commands

  • \? List internal commands.

  • \h List help.

  • \c Connect to a database.

  • \conninfo Current connection information.

  • \l+ List databases.

  • \d+ table_name List tables, views, and sequences.

  • \dt+ List tables.

  • \du+ List users.

  • \q Exit psql.
