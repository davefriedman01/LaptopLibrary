List Conda configuration information. [terminal]

conda info -a

conda config --show

conda config --show-sources

conda config --describe

Update Conda. [terminal]

conda update conda

Update the Conda environment base’s metapackage anaconda. [terminal]

conda activate base && conda update anaconda
